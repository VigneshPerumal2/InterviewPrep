# Visa Fullstack Senior SWE Prep

Beginner-first interview prep site for the Visa Fullstack Senior Software Engineer role.

## Live Site

https://visa-interview-prep.netlify.app

## Local Preview

```sh
python3 -m http.server 5174
```

Then open `http://127.0.0.1:5174`.

## Deployment

This is a static site. Netlify should publish the repository root using the settings in `netlify.toml`.
